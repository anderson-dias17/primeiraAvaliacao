# Primeira avaliação em JAVA

Criar um fluxograma, com teste de mesa e por fim, um programa em java que entre com o nome e quatro notas para 10 alunos. O programa deverá calcular a média aritmética e informar se o aluno está aprovado (media >=7), em recuperação (media >= 5) ou reprovado (media < 5). Ao final, o programa deverá informar o nome e a média do aluno que teve o melhor desempenho e o pior desempenho.
